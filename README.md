# askozia-api

#### Functions
This application connect to askozia api and return console with banch of calls, show next user to hangup and send message in discord to next user.

#### Requiriments
- nodejs
- npm
- yarn

#### How to use
<code>
  git clone git@github.com:jorgediasdsg/askozia-api.git
  cd askozia-api
  yarn
  node src/index.js
</code>
